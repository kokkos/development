\begin{frame}[fragile]{SellMatrix}
  Adding a new matrix format alongside CrsMatrix and BsrMatrix:
  \begin{center}{\ttfamily KokkosSparse::Experimental::SellMatrix}\end{center}
  and support for matrix-vector multiplication with that matrix format, native, cuSPARSE and rocSPARSE implementations available.
  Current limitations:
  \begin{itemize}
  \item only one slice per matrix supported
  \item only single vector allowed for SpMV operation
  \end{itemize}
\end{frame}
