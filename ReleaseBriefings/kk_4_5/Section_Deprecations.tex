  {\Huge Deprecations}

  \vspace{10pt}

\end{frame}

\begin{frame}[fragile]{Macros}
  Deprecating macros
  \begin{enumerate}
  \item that violate c++ reserved \_\_
    \begin{itemize}
    \item \_\_KOKKOSBATCHED\_ENABLE\_INTEL\_MKL\_BATCHED\_\_ replaced by KOKKOSBATCHED\_IMPL\_ENABLE\_INTEL\_MKL\_BATCHED, see PR \href{https://github.com/kokkos/kokkos-kernels/pull/2406}{\#2406}
    \item \_\_KOKKOSBATCHED\_ENABLE\_INTEL\_MKL\_\_ replace by KOKKOSBATCHED\_IMPL\_ENABLE\_INTEL\_MKL, see PR \href{https://github.com/kokkos/kokkos-kernels/pull/2403}{\#2403}
    \item \_\_KOKKOSBATCHED\_PROMOTION\_\_ deprecated and to removed in next major release, see PR \href{https://github.com/kokkos/kokkos-kernels/pull/2392}{\#2392}
    \end{itemize}
  \item that are using the reserved prefix KOKKOS\_
    \begin{itemize}
    \item a lot of {\ttfamily KOKKOS\_COMPONENT\_...} are replace by {\ttfamily KOKKOSCOMPONENT\_...}, see PR \href{https://github.com/kokkos/kokkos-kernels/pull/2372}{\#2372}
    \end{itemize}
  \end{enumerate}
\end{frame}

\begin{frame}[fragile]{General Deprecations}
  \begin{itemize}
  \item {\ttfamily KokkosKernels::TeamBitonicSort} deprecated in favor of: {\ttfamily Kokkos::Experimental::sort\_by\_key\_team}, see PR \href{https://github.com/kokkos/kokkos-kernels/pull/2306}{\#2306}
  \end{itemize}
\end{frame}
