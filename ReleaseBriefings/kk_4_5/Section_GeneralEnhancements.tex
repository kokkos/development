\begin{frame}[fragile]{Common feature}
  Adding functionality to initialize TPLs ahead of using them. This allows users to initialize TPLs right after initializing Kokkos and have cleaner timing results.\\
  Usage:
  \begin{center}
    {\ttfamily KokkosKernels::eager\_initialize();}
  \end{center}
  For more detailed information see PR \href{https://github.com/kokkos/kokkos-kernels/pull/2317}{\#2317}
\end{frame}
